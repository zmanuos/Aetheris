# Aetheris

###### Frontend #####
# Instalacion  
- node v22.14.0

Instalación
1. cd Aetheris
2. npm install

####### Backend #######
## Requisitos
- .NET 8.0 SDK
- MySQL 8.0+

## Instalación
1. Clonar repositorio
2. cd BackendAetheris
3. `dotnet restore` # Instalar dependencias del .csproj
4. Configura `appsettings.json` con credenciales de MySQL 
5. `dotnet ef database update`
6. `dotnet run`
